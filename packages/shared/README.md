# @nfewizard/shared

Core utilities compartilhadas para comunicação com SEFAZ (XML, HTTP, Certificados, Schemas).

## Instalação

```bash
npm install @nfewizard/shared @nfewizard/types
# ou
pnpm add @nfewizard/shared @nfewizard/types
```

## Características

- ✅ **XmlBuilder** - Assinatura digital e geração de SOAP
- ✅ **HttpClient** - Cliente HTTP configurado para SEFAZ
- ✅ **LoadCertificate** - Carregamento de certificados A1
- ✅ **SchemaLoader** - Validação XSD
- ✅ **Utilities** - Funções auxiliares (validações, formatações)
